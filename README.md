# WorkLog PWA

Особистий застосунок для обліку робочого часу та деталей на виробництві.  
Побудовано як PWA — встановлюється на iPhone, працює офлайн, без сервера.

## Функціональність

| Розділ | Опис |
|--------|------|
| ⏱ **ЧАС** | Таймер зміни (старт/стоп), ручне додавання записів, список за сьогодні |
| 📅 **ГРАФІК** | Тижневий розклад змін, навігація між тижнями |
| 📊 **ЗВІТ** | Статистика за тиждень/місяць, понаднормові/нічні години, експорт PDF |
| ⚙️ **ДЕТАЛІ** | Картки деталей з фото (до 5 шт.), нотатками та лічильником виробітку |
| 🔩 **РІЗЬБА** | Довідник різьб та отворів *(у розробці)* |

## Стек

- Vanilla HTML / CSS / JS — без фреймворків
- localStorage — зберігання даних локально на пристрої
- Service Worker — офлайн підтримка та кешування
- jsPDF + jsPDF-AutoTable — генерація PDF звітів
- Neumorphism + Glassmorphism UI — світла тема, базовий колір `#BECAE1`

## Встановлення на iPhone

1. Відкрити посилання Netlify у **Safari**
2. Тап «Поділитися» → «На початковий екран»
3. Застосунок працює як нативний, повністю офлайн

## Деплой на Netlify

### Варіант 1 — Drag & Drop
1. Зайти на [netlify.com/drop](https://netlify.com/drop)
2. Перетягнути папку проєкту в браузер

### Варіант 2 — З GitHub (рекомендовано)
1. Push до GitHub репозиторію
2. Netlify → «Add new site» → «Import from Git»
3. Вибрати репозиторій — деплой автоматичний

## Структура проєкту

```
worklog/
├── index.html      # Весь застосунок (HTML + CSS + JS)
├── manifest.json   # PWA маніфест
├── sw.js           # Service Worker (офлайн кеш)
├── icon-192.png    # PWA іконка
├── icon-512.png    # PWA іконка
├── netlify.toml    # Конфігурація деплою
└── .gitignore
```

## Дані

Всі дані зберігаються виключно в `localStorage` браузера на пристрої користувача.  
Ніякого сервера, ніяких хмарних баз даних — повна приватність.
